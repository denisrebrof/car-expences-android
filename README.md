# car-expenses-android
My little pet-project - app for tracking car expenses.

Kotlin-only, supports multiple data layer / backend implementations: Realm, Room and Firebase.

### Key Features
 - Offline-First
 - Data Sync
 - Clean Architecture
 - Authorization
 - Stats Calculation
 - Expense Filtering

### Stack
 - Kotlin
 - RxJava2
 - Retrofit
 - Room
 - Realm
 - Firebase
 - AAC
